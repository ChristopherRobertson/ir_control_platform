# Operator UI MVP

The active MVP is the generic `Single-Wavelength Pump-Probe` v1 workflow.

Primary navigation is exactly:

1. Session
2. Setup
3. Results

See `docs/ui/single_wavelength_pump_probe_v1_information_architecture.md` for the active UI specification.

Deprecated guidance about broader experiment shells, device dashboards, separate run pages, or advanced/service surfaces is not active for v1.
